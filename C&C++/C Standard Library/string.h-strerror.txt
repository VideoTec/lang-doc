在linux编程中，strerror()是个好东东，
因为一个孤零零的errno看不出个所以然，
然而strerror()返回的错误描述已经给我们解决问题提供了80%的成功率。
但从安全性的角度来讲，strerror_r是更好的选择

#include <string.h>
  char *strerror( int num );
 

功能：函数返回一个被定义的与某错误代码相关的错误信息。

------------------------------
strerror 是标准C库中的函数。
------------------------------
返回的错误信息保存在，errno.h
