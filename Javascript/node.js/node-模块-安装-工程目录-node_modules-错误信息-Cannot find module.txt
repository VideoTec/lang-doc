https://www.npmjs.com/package/node-rest-client

$ npm install node-rest-client

安装后，运行出错：
Cannot find module 'node-rest-client'

问题，必须安装在 【工程所在目录】。
工程所在目录，即，js 文件所在目录。
要把当前目录切换到 【工程所在目录】。

http://stackoverflow.com/questions/9023672/nodejs-how-to-resolve-cannot-find-module-error

Using npm install installs the module into the current directory only 
(in a subdirectory called node_modules). 
Is app.js located under home/dave/src/server/? 
If not and you want to use the module from any directory, 
you need to install it globally using npm install -g.

I usually install most packages locally so that they get checked in along with my project code.
