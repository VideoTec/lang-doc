com.google.common.eventbus
-------------------------------------    
The EventBus allows publish-subscribe-style 
communication between components 
without requiring the components to 
explicitly register with one another (and thus be aware of each other).

com.google.common.io 
-------------------------------------   
This package contains utility methods and classes for 
working with Java I/O; 

for example 
1）input streams, 
2）output streams, 
3）readers, writers, and files.

-------------------------------------  
Arithmetic functions operating on primitive values and BigInteger instances.

-------------------------------------

-------------------------------------

com.google.common.primitives
