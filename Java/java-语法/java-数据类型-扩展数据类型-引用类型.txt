引用类型变量由类的构造函数创建，可以使用它们访问所引用的对象。
这些变量在声明时被指定为一个特定的类型，比如Employee、Pubby等。变量一旦声明后，类型就不能被改变了。

对象、数组都是引用数据类型。

所有引用类型的默认值都是null。

一个引用变量可以用来引用与任何与之兼容的类型。

例子：Animal animal = new Animal(“giraffe”)。
